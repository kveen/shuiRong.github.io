JS构建正则时的一处致命细节，请注意


2016-9-24细节 JavaScript 正则

JavaScript构建正则对象有两种方法：



  • 字面量: /ab+c/i;

  • 构造函数: （还可以细分一下）

    • new RegExp(‘ab+c’, ‘i’);

    • new RegExp(/ab+c/, ‘i’);




当我们构建正则对象时，会有一些特殊字符，需要特殊处理，也就是转义。
JavaScript的转义符和其他语言的一样： \ 反斜杠。
需要注意的是：
当用正则构造函数创造正则对象时，转义需要两个 \ 反斜杠。当然，字面量时一个就够了。
以下两个是等价的：



var re = new RegExp(“\w+”);
var re = /\w+/;
