###写在前面


换用Ubuntu了，首先需要连上校园网，用的是华为的iNode客户端联网。在学校官网下了个 inode-x64 for linux。安装好后提示：MAC地址验证失败。想了想应该是服务端记录的是我上一个系统的MAC地址。然后Google如何在Ubuntu修改MAC地址，根据教程反复修改了 /etc/network/interfaces 里的文件好几次，都没成功。最后在AskUbuntu.com上找到了答案。


##正文


不是命令行，而是图形化操作:


右上角 System Setting –> network –>选择你正在使用的网卡(Wired)–> options(选项)–>Ethernet(以太网)–>Cloned MAC address : ::::(你想要的ＭＡＣ地址)–>重启**
