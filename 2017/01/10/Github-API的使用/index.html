正文

接下来的内容都可以从Github　API文档里得到，实在看不懂英文的也可以去谷歌搜中文翻译版（有人翻译出了中文版）。


获取仓库信息: https://api.github.com/repos/{用户名称}/{仓库名称}
获取仓库根文件下的A文件信息:　https://api.github.com/repos/{用户名称}/{仓库名称}/contents/{A文件名称}
获取A文件下的B文件信息: https://api.github.com/repos/{用户名称}/{仓库名称}/contents/{A文件名称}/{B文件名称}
